# Vibe Coding with Antigravity IDE

The Antigravity IDE is a unified graphical development environment that integrates code editing panels, file explorers, and a built-in terminal with a powerful background AI agent orchestration engine.

## 🚀 How to Use Autonomous Workflows

### 1. Goal Tab Execution
The Goal panel allows you to track and configure autonomous runs in a graphical UI:
1. Open the **Goal** tab in the IDE.
2. In the input box, define the objective:
   ```text
   Read AGENTS.md and execute the next task
   ```
3. Press the **Start** button.
4. You will see a tree of sub-goals and tasks populate dynamically as the agent writes code, compiles, and verifies changes.

### 2. Chat Sidebar Integration
The chat sidebar allows you to converse with the agent workspace assistant:
1. Open the AI chat panel in the sidebar.
2. Type your command referencing the instructions, for example: `"Trigger an enhancement ('e') based on AGENTS.md"`.
3. Review and approve the proposed file edits or terminal commands directly from the chat UI.
